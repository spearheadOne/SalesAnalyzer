# Sales Dashboard

Dashboard to show live and historical sales data.

For the list of endpoints check swagger documentation under /swagger-ui.html

For the healthchecks check /health endpoint.

## Run locally

- To run the application:
```
../gradlew run
```

- To run frontend in dev mode
```
cd dashboard-frontend
yarn dev
```

## Access

Localy dashboard is available at http://localhost:9024 for cloud check terraform output
